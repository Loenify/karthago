# Geplante Paketstruktur #

Idee war drei Pakete für MVC zu verwenden sowie eigene Pakete für die
Datenbank, XML-Schnittstelle und den Verwaltungscode zu verwenden.

Präfix: `org.teamwip.karthago`

- `controller`
- `db`
- `gui`
- `main`
- `model`
- `xml`
